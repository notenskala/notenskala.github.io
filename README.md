# Notenskala Generator – Gymnasiale Oberstufe

## Beschreibung

Dieses Projekt generiert eine an die maximal erreichbare Gesamtpunktzahl angepasste Notenskala gemäß den prozentualen Vorgaben der gymnasialen Oberstufe.

Die Berechnungsgrundlage basiert auf veröffentlichten Prozentwerten:
https://www.abi-rechner.com/wp-content/uploads/2017/04/Tabelle-Oberstufe-Gymnasium-Mit-Prozenten.png

## Implementierung

- Dynamische Skalierung anhand der maximalen Punktzahl
- Prozentuale Schwellenwertberechnung
- Rundungsbasierte Umrechnung auf Ganzzahlen
- Ausgabe der Notenpunkte (0–15)
